To interact with the map there are two tools on the left hand side (the one which is active will have an orange background):

  1. To Select a feature use the select tool.

  2. To Zoom in on an area use the zoom tool.


Note: You CANNOT select a feature to find out information about it if your zoom tool is Active! Also if a feature has a number on it, you must zoom in further to select a feature; the number tells you how many cultural features are located in that general area.

Alternatively you can use the located above the select tool to zoom in or the to zoom out. The button zooms you back to the full extent of Kings County or you can use your mouse wheel to zoom in or out.

To make a popup box disappear click on the red x button in the top right corner or simply select another feature.

Layers can be turned on or off using the checkboxes on the right side of the map.
